# GrainDash Marketing Site

Marketing landing page for GrainDash, served at https://graindash.net

## What's here
- index.html - the entire landing page (self-contained HTML + inline CSS/JS).
- assets/ - image assets (app screenshots used in the device showcase).
- CNAME - custom domain config for GitHub Pages (graindash.net).

## Hosting
Currently hosted on GitHub Pages from the main branch (root).
Any push to main auto-publishes to https://graindash.net

## Editing
This is a static site - no build step. Edit index.html directly and commit.
All CTAs ("Log In", "Get Started", etc.) point to the web app at https://graindashapp.net

## Notes
- The web app (graindashapp.net) is a separate project from this marketing site.
- To move hosting off GitHub Pages, deploy these files as a static site elsewhere
  and repoint the graindash.net DNS accordingly.
